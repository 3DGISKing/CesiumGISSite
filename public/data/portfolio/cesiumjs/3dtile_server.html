Detailed description

Sometimes we need to host 3d tilesets on our server instead of Cesium Ion when the data are sensitive to the restricted public access requirement. It can be implemented by a simple node js express server. But you can notice that the 3d tileset contains nested folders with thousands of small files. It is critical if you plan to download and upload a bunch of 3d tilesets.

One solution would be to use the SQLite format which Cesium team introduces.

This is a simple 3d tile server based on the SQLite format of the 3d tileset. This custom 3d tile server was made with CodeIgniter-3.1.10.

First, you have to convert the 3d tileset to SQLite format to use this server. You can refer to this link for it.

Then copy it and paste it to the “asset” folder of the CodeIgniter.

If tilesets have a more deep level, then add routing rules in application/config/routes.php. We can remove index.php in tileset.json URL by using rewrite rule.
