This project was done by the request of Mr. Joshua Sinclair and its requirements are as follows. Allows drawing of points, polyline, and polygons. Allows editing of the style, i.e polyline color, polyline width, polygon outline color, polygon outline width, polygon fill color, clamped to 3d tile. Users can select the already drawn feature and then change its style and also change the style during the drawing. Save the drawn points, polyline and polygons to a simple MySQL database Geojson string. When the application starts at first time, it must load points, polylines and polygons from the database.
