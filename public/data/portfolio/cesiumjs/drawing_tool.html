Job description

  1. Allows drawing of points, polyline, and polygons.
  2. Allows editing of the style, i.e polyline color, polyline width, polygon outline color, polygon outline width, polygon fill color, clamped to 3d tile. Users can select the already drawn feature and then change its style and also change the style during the drawing.
  3. Save the drawn points, polyline and polygons to a simple MySQL database as a GeoJSON string.
  4. When the application starts at first time, it must load points, polylines and polygons from the database.

Detailed drawing logic:

Step1:

User clicks the scene to draw the first point.

step1

Step2:

After the first click, a popup like the above will appear. Clicking the first drawn vertex or a green tick will exit this tool and save the point as GeoJSON format to MySQL database. Clicking the red X icon will remove the first drawn vertex and reset the drawing. At this time a user would be ready to restart step1.

step2

Step3:

After the second click, the popup same as step2 shows at vertex 2. Clicking the vertex 2 or a green tick will exit this tool and save the line as GeoJSON format to MySQL database. Clicking red X will remove the vertex 3 and reset the drawing and return the user to step2.

step3

Step4:

After the third click, show popup at vertex 3. Clicking vertex or a green tick will exit this tool and save the polyline as GeoJSON format to MySQL db. Clicking red X will remove the vertex3 and reset the drawing and return the user to step3.

step4

To draw a polygon, the user clicks the first vertex and then the popup will show, so the user can complete the drawing of the polygon or return the previous state.
