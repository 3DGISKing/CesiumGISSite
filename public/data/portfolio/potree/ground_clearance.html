Job description

It measures the vertical distance to the ground by clicking on one point (without the need to click on the ground).
The objective of this request is to use it for any elevated cables (i.e. powerlines, telephone, etc) or infrastructures (bridges).
We want to have a quick way of measuring the height of one point above the ground.
In case of cables, it is very likely to have ground below the cable, but we can make things easier for this request
and calculate the height above the first object you find in the vertical projection of the point.
When you click on a point, the tool draws the vertical line from the point to the ground and returns the height.
